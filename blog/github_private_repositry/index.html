GitHubのPrivateリポジトリの上限が無くなった

May 12, 2016

https://github.com/blog/2164-introducing-unlimited-private-repositories

GitHubのPrivateリポジトリの上限が無くなったので、BitBucketに置いていた個人用リポジトリを月額7ドルでGitHubに置けるようになった。このブログ用のリポジトリも移動しよう。

  • GitHubにリポジトリを作成
  • gitのリモートの向き先を変更
  • git push
  • werckerに追加
  • このページをcommitしてpush

werker.ymlはそのまま使える。werkerにbuildとdeployのWorkflowなどを追加。このファイルをcommitしてpushすれば、werckerがGitHub Pages用リポジトリに反映してくれて完了。（になると思う）
